# AI Career Chatbot

## Overview

AI Career Chatbot is a Python-based chatbot that provides information about Artificial Intelligence, Machine Learning, Deep Learning, Python, GitHub, internships, and career guidance.

## Features

* Answers AI-related questions
* Provides career guidance
* Interactive chat interface
* Beginner-friendly chatbot

## Technologies Used

* Python
* Dictionaries
* Loops
* Conditional Statements

## Sample Questions

* What is AI?
* What is Machine Learning?
* What is CNN?
* How can I learn Python?
* What is GitHub?
* How can I get an internship?

## Future Improvements

* Add Gemini API integration
* Add NLP capabilities
* Create a web interface using Streamlit
